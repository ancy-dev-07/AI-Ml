Git & GitHub Documentation

What is Git?

  • Git is a Version Control System
  • Tracks code changes efficiently
  • Helps multiple developers work together

What is GitHub?

GitHub is a web-based Git repository hosting service used to store, manage, and collaborate on projects.

GitHub Alternatives

  • Bitbucket
  • GitBucket

Creating a New Repository on GitHub

  1. Click New Repository
  2. Enter repository name
  3. Click Create Repository
  4. Choose New File or Upload Existing File
  5. Write content and click Commit Changes

Installing & Configuring Git

Check Git installation:

git

Set username and email:


git config --global user.name "ancy-dev-07"
git config --global user.email "ancys722@gmail.com"

Cloning a GitHub Repository in VS Code

  1. Create a folder (Desktop or C Drive)
  2. Open the folder in VS Code
  3. Open Terminal
  4. Copy repository link from GitHub
git clone <repository-link>
cd repository-folder

Git Workflow

Check file status:

git status

Add file to staging:

git add test2.txt

Commit changes:

git commit -m "Added test2.txt"

Push to GitHub:

git push origin main

Flow: Insert ‚Üí Add ‚Üí Commit ‚Üí Push

File Status Indicators

  • U ‚Üí Untracked file
  • M ‚Üí Modified file

Use the same workflow: add ‚Üí commit ‚Üí push

Pulling New Files from GitHub

If files are added directly in GitHub:

Adding a New Folder to GitHub from VS Code

Initialize Git:

git init

Add and commit:


git add .
git commit -m "Initial commit"

Create repository in GitHub and connect:

git remote add origin <repository-link>

Check branch:

git branch

Rename branch:

git branch -M main

Push to GitHub:

git push origin main
