## what is Data structure

A Data Structure is a way of organizing and storing data in a computer so that it can be used efficiently.

Simple meaning:

Data Structure = How data is stored and arranged

Example:

- Storing marks in a list

- Storing student details using dictionary

- Storing items in a queue (like a line)

## What is an Algorithm?
Definition:

An Algorithm is a step-by-step procedure to solve a problem.

- Simple meaning:

Algorithm = How to solve a problem step by step


##### Example (Real life):

- Making tea

- Boil water

- Add tea powder

- Add sugar

- Add milk

- Serve

 This is an algorithm.

#### Why the name?

Named after Al-Khwarizmi, a mathematician

#### Example:
Steps to find largest number in a list.


| Data Structure          | Algorithm                   |
| ----------------------- | --------------------------- |
| Stores data             | Processes data              |
| Focuses on organization | Focuses on logic            |
| Example: Array          | Example: Searching, Sorting |

#### Simple Example Combining Both
```
# Data Structure: List
numbers = [10, 5, 20, 8]

# Algorithm: Find maximum number
max_num = numbers[0]

for num in numbers:
    if num > max_num:
        max_num = num

print(max_num)
```
